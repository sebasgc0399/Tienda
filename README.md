# Tienda

Vitrina e-commerce-lite para un negocio familiar en Colombia: catálogo de arreglos artesanales, gorras y decoraciones, con carrito y checkout por WhatsApp (sin pasarela de pago) y panel de administración para la dueña.

Stack: Next.js 16 (App Router) · Supabase (Postgres + Auth + Storage) · Tailwind CSS v4 + shadcn/ui · Vercel.
